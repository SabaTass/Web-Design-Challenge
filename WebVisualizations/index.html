Summary

Data Analysis

The purpose of this project was to analyze how weather changes as one moved away from or get closer to the equator.

The data set is used from cities.csv which is a collection of 500 random cities pulled from Openweather API

After collecting and assembling the dataset from random cities, Matplotlip was used to plot various graphs to show Max-temperature, cloudiness, wind speed, and humidity.

This website shows a complete analysis and comparison between weather trends and locations of different cities.
